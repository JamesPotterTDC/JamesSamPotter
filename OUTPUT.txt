RAILWAY SERVICE COMMANDS (Copy to Railway Dashboard)
================================================================================

Service 1: backend-web
  Root Directory: backend
  Start Command:
python manage.py migrate && python manage.py collectstatic --noinput && gunicorn cycling_dashboard.wsgi:application --bind 0.0.0.0:$PORT --workers 4

Service 2: backend-worker  
  Root Directory: backend
  Start Command:
celery -A cycling_dashboard worker --loglevel=info --concurrency=2

Service 3: backend-beat
  Root Directory: backend
  Start Command:
celery -A cycling_dashboard beat --loglevel=info --scheduler django_celery_beat.schedulers:DatabaseScheduler

Service 4: frontend (optional, or use Vercel)
  Root Directory: frontend
  Build Command: npm ci && npm run build
  Start Command: npm start

ENV VARS CHECKLIST (Set in Railway for All Backend Services)
================================================================================

Generate these first:
python -c "from django.core.management.utils import get_random_secret_key; print(get_random_secret_key())"
python -c "from cryptography.fernet import Fernet; print(Fernet.generate_key().decode())"
openssl rand -hex 32
openssl rand -hex 32

Then set:
  DJANGO_SECRET_KEY=<generated-50-chars>
  ENCRYPTION_KEY=<generated-fernet-key>
  ADMIN_API_TOKEN=<generated-hex>
  STRAVA_WEBHOOK_VERIFY_TOKEN=<generated-hex>
  STRAVA_CLIENT_ID=<from-strava-api-settings>
  STRAVA_CLIENT_SECRET=<from-strava-api-settings>
  ALLOWED_HOSTS=your-backend.railway.app
  FRONTEND_URL=https://your-frontend.vercel.app
  CORS_ALLOWED_ORIGINS=https://your-frontend.vercel.app
  CSRF_TRUSTED_ORIGINS=https://your-frontend.vercel.app
  DEBUG=False
  MAP_REDACT_PERCENT=10
  STRAVA_FETCH_STREAMS=False

Auto-provided by Railway (don't set):
  DATABASE_URL, REDIS_URL, PORT

Frontend env var (Vercel):
  NEXT_PUBLIC_API_URL=https://your-backend.railway.app/api
